# Portfólio Ana Flávia

Portfólio profissional com layout **retro** (terminal) e **moderno**.

## Comandos do terminal (layout retro)

No rodapé da tela, use o prompt `> ` para navegar ou iniciar o download.

### Navegação

| Comando     | Vai para     |
|------------|--------------|
| `home` ou `inicio` ou Enter vazio | Início (`/`) |
| `sobre` ou `sobre-mim`            | Sobre mim (`/sobre-mim`) |
| `projetos`                       | Projetos (`/projetos`)   |
| `experiencias`                   | Experiências (`/experiencias`) |
| `contatos`                       | Contatos (`/contatos`)   |

### Xmodem – download do portfólio (modo moderno)

Para “baixar” o portfólio e mudar para o layout moderno:

- **`xmodem`** — inicia o download simulado (barra de progresso).
- **`xmodem portfolio.zip`** — mesmo efeito; o nome do arquivo é apenas ilustrativo.

Ao concluir o “download”, a interface passa automaticamente para o **layout moderno**.
